# 🚀 Deadline Guardian AI

## Problem Statement

Students and professionals often struggle to manage multiple deadlines, estimate task risk, and avoid burnout. Existing planners create schedules but do not provide intelligent guidance, risk analysis, or productivity insights.

## Solution

Deadline Guardian AI is an AI-powered productivity assistant that transforms user tasks into actionable schedules while predicting risks, monitoring burnout, generating study strategies, and providing productivity analytics.

## Features

* 📌 Task Extraction using AI
* ⏳ Smart Hour Allocation
* 🔥 Deadline Risk Prediction
* 🧠 Burnout Detection
* 📈 Productivity Forecast
* 🎯 Success Probability Prediction
* 🚨 Smart Notifications
* 🌙 Focus Session Generator
* 📋 AI Weekly Report
* 📄 PDF Report Export
* 📅 Calendar Export
* 💬 Guardian AI Chat Assistant
